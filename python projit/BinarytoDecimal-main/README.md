<b>Android_PreBuild</b>
<br>
<br>
Contains the mobile version of the app, including:
<br>
1. Android Debug Bridge for Windows == 1.0.39 + dlls
2. Buildozer.spec + requirements
3. main.py (edited mobile version)
4. logo.png
